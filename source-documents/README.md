# Source Documents

This folder holds the canonical Word documents for *Intentional Journey of the
Heart*, organized by version.

The Markdown content rendered on the reading site (under `docs/`) is derived
from these files. When in doubt, **the `.docx` files in this folder are the
source of truth**.

## Versions

- [`v5_5_7/`](v5_5_7/) — Current working version (March 2026, Revised Edition).
  Adds Volume 6 to the previously published five-volume structure.

## File naming

`IJH_<section>_<version>.docx`

- `Intro` — Introduction
- `Vol1` through `Vol6` — the six volumes
- `Read Me First` — orientation document

Companion `vol<N>-claims.yml` files contain the claims data extracted from
each volume for cross-referencing.
