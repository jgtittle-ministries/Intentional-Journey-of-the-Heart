# Contributing

Thank you for your interest in *Intentional Journey of the Heart*.

This project welcomes engagement, but the editorial direction rests with the
author, **John G. Tittle**. The volumes, formation documents, and companion
materials are personal work — please reach out before opening substantial pull
requests.

## Ways to contribute

- **Corrections** — typos, broken links, scripture reference errors. Open an
  issue or a small pull request.
- **Questions and discussion** — open an issue. Conversations about the ideas
  in the volumes are welcome.
- **Response papers, case studies, extensions** — these are on the roadmap.
  Please open an issue first to discuss scope before drafting.
